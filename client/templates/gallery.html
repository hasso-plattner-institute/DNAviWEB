Gel image processing



1. Band detection



Example of DNAvi output after the user has submitted a DNA gel image: the detected DNA bands will be shown as color-coded masks on the thresholded input image. In this example, we can nicely see that all DNA bands are covered by a colored mask, indicating that all bands have been correctly identified.





2. Lane borders



Next, DNAvi will use this information to infer gel lane borders. This is an example output plot generated by DNAvi, helping you to check whether lane borders have been correctly inferred. In this example, we can nicely see that all DNA lanes have the same size, and that all DNA bands fall inside a lane, indicating that all lanes have been correctly identified.





3. Intensity profile



Finally, DNAvi will calculate a profile line for each lane. To allow you to understand if this process was correctly executed, DNAvi plots the gel lane, the central profile line, and the resulting profile into a pdf plot. In this example, we can see that the lower and upper marker at y-positions 0 and ~410 have been correctly inferred, and so has the broader DNA band at y-position ~300.




Ladder quality control



Peak detection



Example of DNAvi output after a DNA ladder intensity profile has been analyzed for peaks. We can see that 11 peaks have been detected in this ladder. Now, DNAvi will compare this to the user-provided file. If all 11 peaks (orange crosses) have been annotated by the user with basepair information, the quality control is considered as passed and downstream processing continues.




DNA fragment size plot



Per-sample profile plot



Example of DNAvi output after the quality control for the DNA ladder was passed and basepair positions were annotated. We can see 3 samples' Dna fragment length profile, showing a peak centered at around 150 bp, and a smaller second peak at ~250 bp. These are likely to be the mono- and di-nucleosomal cell-free DNA fraction. All exact position values are also provided as a source data table, so that plots can be edited and customized if needed. The default plot format pdf, i.e. publication quality vector graphics.




DNA fragment size plot



Per-sample profile plot





Example of DNAvi output for a set of grouped samples (group information provided by the user through a metadata file). We can see that the average fragment size of samples in the Control group is smaller than in the treatment group.
